# minimal newtab

a minimal, private and lightweight new tab page without any bloat.

## features

- multiple search engines (duckduckgo, google, brave)
- scroll or click to switch engine
- customizable text and background colors
- localstorage based settings
- no external dependencies
- no tracking
- no analytics
- no permissions required

## install as extension

1. download most recent extension release
2. extract .zip release file to folder
3. go to chrome://extensions
4. enable developer mode
5. click "load unpacked"
6. select the extracted folder

## privacy

this project does not collect, store or transmit any personal data.
all settings are stored locally in your browser.
